CHIT MASTER - Online Chit Fund Management System

A full-stack web application designed to digitalize traditional chit fund operations by providing a transparent, efficient, and user-friendly platform. The system supports **multi-user access**, **real-time bidding**, **premium tracking**, **kasar (balance) calculation**, and **end-to-end chit cycle management**—replicating real-world chit fund workflows.

Features:
  User & Admin Modules
- User Registration & Login
- Role-based Access (User/Admin)
- Secure Authentication

  Chit Fund Operations
- Chit Scheme Creation & Enrollment
- Monthly Premium Payments
- Live Bidding System (Real-Time Updates)
- Automated Kasar Calculation
- Real-time Chit Progress Tracking

  Dashboard & Tracking
- User Dashboard showing:
  - Active chits
  - Payment history
  - Bidding status

- Admin Dashboard for:
  - Managing users
  - Adding chit schemes
  - Monitoring ongoing auctions

  Transactions & Management

Objective:
To provide a digital chit fund management platform that ensures:

- Transparent bidding and settlements  
- Reduced operational cost  
- Faster processes compared to manual chit systems  
- Better data accuracy & user convenience  

Concept:
A chit fund is a savings system where a group contributes a fixed monthly amount.  
Every month, a bidding auction is conducted, and the highest bidder receives the chit amount after deducting the foreman fee.

This system automates:
- Monthly contributions  
- Bidding  
- Winner selection  
- Payment & balance updates  
- Kasar calculation
